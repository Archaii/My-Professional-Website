6. MENU & SUBMENU INTERACTIONS
================================================================================

PRIMARY NAVIGATION BEHAVIOR:
----------------------------

DESKTOP VIEW:
1. Hover over "Profile" or "Portfolio" menu items
   → Submenu dropdown appears below
   → Submenu remains visible while mouse is over menu or dropdown

2. Click submenu item
   → Navigates to selected page
   → Submenu closes
   → Selected page indicated with aria-current="page"

3. Click "Contact" menu item
   → Navigates to homepage contact section (index.html#contact)
   → Smooth scroll to contact form

MOBILE VIEW:
1. Click hamburger menu icon (three horizontal bars)
   → Full navigation menu slides in or expands
   → Menu toggle icon changes (animated)

2. Click "Profile" or "Portfolio" text or toggle button
   → Submenu expands below parent item
   → Arrow indicator rotates to show expanded state
   → Aria-expanded attribute changes to "true"

3. Click submenu item
   → Navigates to selected page
   → Mobile menu closes automatically

4. Click outside menu or close button
   → Mobile menu collapses/hides
