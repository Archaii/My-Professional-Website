Code Snippet - Conversion Logic

Here's the core PHP function that handles length conversions:


<?php
function convertLength($value, $fromUnit, $toUnit) {
    // Conversion rates to meters (base unit)
    $toMeters = [
        'meter' => 1,
        'kilometer' => 1000,
        'centimeter' => 0.01,
        'millimeter' => 0.001,
        'mile' => 1609.34,
        'yard' => 0.9144,
        'foot' => 0.3048,
        'inch' => 0.0254
    ];
    
    // Convert to meters first, then to target unit
    $meters = $value * $toMeters[$fromUnit];
    $result = $meters / $toMeters[$toUnit];
    
    return round($result, 4);
}
?>
